# AdmitHub

* This website is a Admission Related related website.
* In home page We can Search university , Show Top 3 University and there details, Graduate Student , Research Paper, Review Section
*  In college tab we can see all the University
* In Admission we see all college and after click the college name we can fill the college application from
* After admission we can see the data in My collage section
